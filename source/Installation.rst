Installation
============
Spider package is mainly based on python.

Software dependencies
---------------------
.. code-block:: python

   anndata
   matplotlib
   numba
   numpy
   pandas
   scanpy
   scikit-learn
   scipy
   seaborn
   squidpy
   plotly
   joblib



Installation
---------------------
Install Spider via PyPI by using:

.. code-block:: shell

   pip install st-spider
